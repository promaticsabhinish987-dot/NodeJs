## ✅ HTTP Methods (Short Answer)

* **GET** → Fetch data

  * Idempotent ✅
  * No body

* **POST** → Create resource

  * Idempotent ❌
  * Has body

* **PUT** → Replace entire resource

  * Idempotent ✅
  * Full data required

* **PATCH** → Update partial fields

  * Usually idempotent ⚠️
  * Partial data

* **DELETE** → Remove resource

  * Idempotent ✅

---

## 🔁 Idempotency (1 line)

> Same request multiple times → same result.

---

## 🎯 One-line summary (say this)

> “GET fetches, POST creates, PUT replaces, PATCH updates partially, DELETE removes — and idempotency helps decide safe operations.”
